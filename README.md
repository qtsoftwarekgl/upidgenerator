# CRVS-USER-SERVICE-API

API with [Node.js](http://nodejs.org/), [Expressjs] (https://expressjs.com/) and [mongoose](https://mongoosejs.com)
## Requirements

1. Node >= 10.16.0

2. NPM >= 6.9.o

3. GIT >= 2.23.0

### Create Database

1. Create Database manually

### Local Server Setup

1. Make sure you have [Node.js](http://nodejs.org/) installed.

2. Make sure you have updated .env file

```bash
git clone https://gitlab.com/crvs-systems-backend/users-services-apis.git
cd users-services-apis
npm install
npm start
```

Your app should now be running on [ localhost:3000 ](http://localhost:3000/).
